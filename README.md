# Интерактивная карта предприятий Росатома

Академический и интерактивный веб-проект, разработанный студентами **Томского политехнического университета (ТПУ)** в рамках практической работы Инженерной школы ядерных технологий.

Приложение представляет собой двухконтурный интерактивный картографический стенд, предназначенный для наглядного отображения географического расположения, описания и связей с ТПУ ключевых предприятий Государственной корпорации «Росатом», научно-образовательных баз и совместных проектов.

---

## 🚀 Основной функционал

1. **Гибридный картографический движок:**
   * **OpenStreetMap (Leaflet):** Свободная векторная карта, работающая без ключей и ограничений. Координаты и масштабирование жестко зафиксированы в пределах Евразии и территории РФ, чтобы предотвратить случайный уход экрана в океанические зоны.
   * **Яндекс.Карты API 2.1:** Высокодетализированная коммерческая подложка со встроенной кластеризацией и быстрой онлайн-загрузкой.
2. **Панель администрирования (Admin Panel):**
   * Защищена кодом авторизации: по умолчанию **`201721`** (константа в коде).
   * Вызывается из любого места приложения по горячей комбинации клавиш **`Ctrl + Alt + A`**.
   * Полнофункциональное управление базами данных в реальном времени: добавление, удаление, динамический поиск по названию.
3. **Двухсторонний импорт и экспорт (SheetJS / JSON):**
   * Полное резервное копирование и восстановление всей базы данных в формате структурированного **JSON**-файла.
   * Импорт и экспорт таблиц **Excel (.xlsx)** прямо в браузере (на клиенте) с автоматическим скачиванием и валидацией пустого Excel-шаблона.
4. **Облачная синхронизация (Firebase Sync):**
   * Автоматическая фоновая синхронизация измененных данных через Firebase Firestore для обеспечения связности данных на нескольких выставочных терминалах.
   * Устойчивый локальный оффлайн-режим (автосохранение в `localStorage` при сбоях сети).
5. **Автоматический ночной спящий режим (Sleep Mode):**
   * Скрипт интеллектуального определения времени (активен с **22:00 до 06:00**).
   * Автоматически накладывает темную энергосберегающую экранную заставку с часами и пульсирующей анимацией, предохраняющую OLED и LCD панели выставочного стенда от выгорания пикселей при статичном изображении. Заставка разблокируется простым кликом.

## 🛠 Технологический стек

* **Frontend:** React 18 / 19, TypeScript, Tailwind CSS, Lucide Icons, Framer Motion (анимации).
* **Сборка и плагины:** Vite, `vite-plugin-singlefile` (для возможности компиляции всего приложения в один полностью портативный HTML-файл для флеш-накопителей).
* **Гео-сервисы:** Leaflet Library (OSM), Yandex Maps API v2.1.
* **Работа с таблицами:** SheetJS (библиотека `xlsx`).
* **База данных:** Firebase Firestore / Google Cloud Storage, LocalStorage.

## 📋 Инструкция по запуску

### В режиме разработки
1. Установите зависимости проекта:
   ```bash
   npm install
   ```
2. Запустите локальный dev-сервер:
   ```bash
   npm run dev
   ```
3. Откройте приложение в браузере по адресу `http://localhost:3000`.

### Сборка в Production
* Собрать дистрибутив:
  ```bash
  npm run build
  ```
  Команда компилирует оптимизированные файлы сборки в каталог `dist/`.
